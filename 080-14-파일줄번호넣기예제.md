# 파일 열어서 줄번호 넣기

## C# 코드
```csharp
namespace  GrammarTest
{
    public class GrammarTest
    {
        
        public static void Main(String[] args)
        {
            var filePath = "/Users/jeongjunghwan/Downloads/qcustomplot.h";
            if (File.Exists(filePath))
            {
                var lines = File.ReadLines(filePath, Encoding.UTF8)
                    .Select((s, ix) => $"{ix + 1,4} : {s}").ToArray();
                foreach (var line in lines)
                {
                    Console.WriteLine(line);
                }
            }
        }
    }
}
```

## 📌 C++23 버전
```cpp
#include <iostream>
#include <fstream>
#include <string>
#include <format>

int main() {
    std::string filePath = "/Users/jeongjunghwan/Downloads/qcustomplot.h";
    std::ifstream file(filePath, std::ios::binary);
    if (!file) {
        std::cerr << "파일을 열 수 없습니다.\n";
        return 1;
    }

    std::string line;
    int lineNumber = 1;
    while (std::getline(file, line)) {
        std::cout << std::format("{:4} : {}\n", lineNumber++, line);
    }
}
```

- std::format은 C++20 이상에서 사용 가능.
- UTF-8 파일을 읽을 때는 소스 파일 인코딩이 UTF-8이면 그대로 출력 가능.

## 📌 Java 버전
```java
import java.io.IOException;
import java.nio.file.*;
import java.nio.charset.StandardCharsets;
import java.util.List;

public class GrammarTest {
    public static void main(String[] args) throws IOException {
        String filePath = "/Users/jeongjunghwan/Downloads/qcustomplot.h";
        Path path = Paths.get(filePath);

        if (Files.exists(path)) {
            List<String> lines = Files.readAllLines(path, StandardCharsets.UTF_8);
            for (int i = 0; i < lines.size(); i++) {
                System.out.printf("%4d : %s%n", i + 1, lines.get(i));
            }
        }
    }
}

```
- Files.readAllLines로 UTF-8 읽기.
- printf로 줄 번호 포맷.

## 📌 Python 버전
```python
from pathlib import Path

file_path = Path("/Users/jeongjunghwan/Downloads/qcustomplot.h")

if file_path.exists():
    with file_path.open(encoding="utf-8") as f:
        for idx, line in enumerate(f, start=1):
            print(f"{idx:4} : {line.rstrip()}")
```

- enumerate로 줄 번호 자동 증가.
- rstrip()으로 줄 끝 개행 제거.
